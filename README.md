# project-manager-kanban

## Description
- A kanban-board based project-management tool.
- Responsive UI.
- Tasks and columns can be dragged.
- Role-based users.
- JWT authentication.

### Technologies
- Vue 3
- Vue Router
- Composition API
- Tailwind CSS

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
